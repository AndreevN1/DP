Цели сайта:
  1. Создать целостное и профессиональное впечатление.
  2. Наглядно представить суть проекта.
  3. Продемонстрировать процесс, а не только результат.
  4. Обеспечить легкий и быстрый доступ к ключевым выводам.
  5. Выступить в роли мощного визуального сопровождения для устной речи.
Целевая аудитория:
  1. Курс: выпускной курс.

Пример структуры HTML-кода, демонстрирующий принцип вложенности тегов:
<div class="top-section">
  <!-- Шапка сайта -->
  <header>
    <div class="logo">
      <img src="лого.jpg" alt="Логотип">
      <div class="college-name">
        ГБПОУ «Пермский политехнический колледж...
      </div>
    </div>
  </header>

  <!-- Навигационное меню -->
  <nav>
    <div class="nav-container">
      <ul>
        <li><a href="index.html">ГЛАВНАЯ</a></li>
        <!-- Остальные пункты меню -->
      </ul>
    </div>
  </nav>
</div>

<!-- Основной контент -->
<main>
  <div class="content">
    <div class="floating-frame">
      <h1>Заголовок страницы</h1>
      <p>Содержимое страницы...</p>
    </div>
  </div>
</main>
Комментарии к коду:
• Теги вложены друг в друга для создания иерархической структуры
• Классы используются для стилизации элементов через CSS
• Комментарии помогают ориентироваться в коде
• Семантические теги (header, nav, main) улучшают читаемость
